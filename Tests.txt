1. Добавить в корзину футболку Govno Black
	-> Перейти на главную страницу
	-> Жмём на кнопку "Футболки"
	-> Выбираем футболку Govno Black
	-> Выбираем размер S
	-> Жмём добавить в корзину
Ожидаемый результат: В корзину добавилась футболка Govno Black размер S стоймостью 1450 руб. с ценой доставки 300 итого 1750 руб.

2. Добавить первый рекомндуемый товар(Термокружка) в корзину:
	-> Нажимаем кнопку "Продолжить покупку"
	-> Жмём на первый рекомендуемый товар 
	-> Нажимаем кнопку "Добавить в корзину"
Ожидаемый результат: В корзину добавилась добавилась Термокружка стоймостью 1500 руб. с ценой доставки 300 итого 3250 руб

3. Добавить ещё 4 футболки Govno Black к покупке
	-> Переходим в корзину 
	-> Жмём кнопку + у пункта Govno Black 4 раза
Ожидаемый результат: В корзину добавились 4 футболки Govno Black размер S стоймостью 1450 руб. с ценой доставки 300 итого 10 250 руб.

4. Удалить 2 футболки Govno Black из корзины
	-> Переходим в корзину 
	-> Жмём кнопку - у пункта Govno Black 4 раза 
Ожидаемый результат: Из корзины удалились 2 футболки Govno Black размер S стоймостью 1450 руб. с ценой доставки 300 итого 6 750 руб.

5. Приметить промокод White
	-> Переходим в корзину 
	-> Вводим в поле промокод "White"
	-> Жмём кнопку "Применить"
Ожидаемый результат: Скидка 15% на все чёрные вещи(Только на футболки Govno Black), т.е. -787,5 руб со стоймости заказа. Итого 5 962,5 руб.

6. Ввод не валидный данных при оформлении заказа
	-> Переходим в корзину 
	-> Жмём кнопку "Оформить заказ"
	-> Записываем в поля Имя, Отчество, Фамилия, Телефон, Эл. Почта, Страна, Город, Край/Область/Регион, Улица/Дом/Квартира, Почтовый индеекс текст "123"
	-> Жмём кнопку "Далее"
Ожидаемый результат: Сообщение ошибки по Эл. Почта.

7. Оформление заказа с невалидной картой
	-> Переходим в корзину 
	-> Жмём кнопку "Оформить заказ"
	-> Записываем валидную информацию в поля Имя, Отчество, Фамилия, Телефон, Эл. Почта, Страна, Город, Край/Область/Регион, Улица/Дом/Квартира, Почтовый индеекс
	-> Жмём кнопку "Далее"
	-> Выбираем карту безналичный расчёт картой Visa
	-> Жмём кнопку "Оплатить"
	-> После переадресации на сайт кассы, жмём кнопку "Оплатить"
	-> Вводим невалидные данные для карты Visa
	-> Жмём кнопку "Оплатить"
Ожидаемый результат: Олата не прошла. Корзина пуста.
